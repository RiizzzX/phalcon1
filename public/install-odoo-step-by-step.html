4 Install Modules (URUTAN PENTING!)

Install modules dengan urutan berikut:

  1. Accounting
    • Hapus filter "Apps" di search bar
    • Search: accounting
    • Klik Install pada module "Accounting"
    • Tunggu sampai selesai (bisa 2-3 menit)
  2. Inventory
    • Search: inventory
    • Install module "Inventory"
    • Tunggu sampai selesai
  3. Purchase
    • Search: purchase
    • Install module "Purchase"
    • Tunggu sampai selesai
  4. Sales
    • Search: sales
    • Install module "Sales"
    • Tunggu sampai selesai

5 Verify Installation

Setelah semua module terinstall, verify dengan cara:

  • Klik menu Apps
  • Filter dengan "Installed"
  • Pastikan ada: Accounting, Inventory, Purchase, Sales
